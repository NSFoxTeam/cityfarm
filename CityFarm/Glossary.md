# Глоссарий гидропонных терминов

> Справочник доменных терминов для коррекции STT (speech-to-text) ошибок в расшифровках курса по городскому фермерству. Колонка **STT-варианты** содержит наблюдаемые ошибки распознавания речи, которые должны быть заменены на правильный термин при обработке транскриптов.
>
> Глоссарий будет расширяться по мере обработки новых модулей в Фазе 2.
