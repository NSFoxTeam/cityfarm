## Параметры

### Преимущества светодиодов

| Параметр | Значение | Примечание |
|----------|----------|------------|
| Световая отдача (белые LED) | 170 лм/Вт | При цветовой температуре 4000 К, CRI 85 |
| Фотосинтетическая фотонная эффективность | до 4,5 мкмоль/Дж | Для цветных (красных) светодиодов |
| КПД (цветные красные LED) | до 76% | — |
| Срок службы | до 50 000 часов | Критерий L70 |
| Рост параметров при низких температурах | да | Особенно для красных и желтых LED |
| Управление спектром | да | Комбинирование разных чипов |
| Стабилизация мощности при скачках напряжения | да | — |
| Размещение в зоне растений | да | Низкая температура излучающей поверхности |
| ИК-излучение в фитооблучателях | минимальное | — |
| Безопасное напряжение | < 40 В | Безопасно для человека |
| Экологическое качество | высокое | Нет ртути (в отличие от ДРЛ/НЛВД) |

### Недостатки светодиодов

| Недостаток | Следствие |
|-----------|----------|
| Низкая единичная мощность | Требуется большее количество LED-элементов |
| Необходимость радиатора | Увеличивает габариты конструкции |
| Деградация при нагреве | Снижение параметров и срока службы при высокой температуре окружающей среды |
| Большие габариты и масса | Для световых приборов со светодиодами |
| Высокая стоимость | Начальные инвестиции выше, чем у НЛВД |
