## Суть

Вентиляция обеспечивает приток свежего воздуха, контроль температуры, влажности и концентрации CO2. Особенно важна в рабочих зонах (приготовление раствора, фасовка, засеивание) и зонах с постоянным присутствием сотрудников.

## Детали

**Функции вентиляции:**
1. Регулирование температуры -- удаление тёплого воздуха, подача прохладного
2. Контроль влажности -- удаление избыточного влажного воздуха
3. Контроль концентрации CO2

**Температурные нормы (СанПиН) по нагрузке:**

| Категория труда | Допустимая температура |
|----------------|----------------------|
| Лёгкая работа | до 28 C |
| Умеренная | 18-27 C |
| Тяжёлая | 15-26 C |

**Рекуператор:** устройство в системе вентиляции, которое нагревает/охлаждает входящий воздух за счёт выходящего. Снижает теплопотери зимой, предотвращает перегрев летом.

**CO2 для растений:**
- Атмосферный уровень: ~400 ppm (достаточно для базового фотосинтеза)
- Оптимальный: до 1200 ppm (активное развитие, быстрое наращивание биомассы)
- Для сотрудников: при >600 ppm -- недомогание, снижение работоспособности (ГОСТ 34941-11)

**Два подхода к CO2:**
1. Вентиляция зоны выращивания (приток атмосферного CO2 ~400 ppm) -- без баллонов
2. Баллонный CO2 + герметизация зоны выращивания (поддержание 1200 ppm) -- избегать вентиляции для предотвращения утечки газа

**Проектирование воздуховодов:**
- Вывод через ближайшую внешнюю стену или крышу
- Чем короче путь до улицы -- тем дешевле монтаж и эффективнее работа
- Длинные воздуховоды: повышенное сопротивление, мощные вентиляторы, больше шума, скопление влаги
- Длинные участки необходимо теплоизолировать
- Вывод воздуха -- подальше от окон и дверей

**Кондиционирование:**
- Наружный блок ближе к внутреннему (короткая трасса = меньше потерь)
- Длинные трассы удорожают монтаж, снижают эффективность
- Учитывать перепад высот между блоками
- Наружный блок -- в хорошо вентилируемом месте с доступом для обслуживания
