## Суть

Питательные элементы взаимодействуют с растительным организмом по трём механизмам: аддитивность (суммарный эффект), синергизм (взаимное усиление) и антагонизм (взаимное подавление). Понимание этих взаимодействий критически важно для составления сбалансированного питательного раствора.

## Детали

### Аддитивность

Действие смеси элементов равно сумме действия каждого элемента в отдельности. Растения поглощают минеральные элементы по закону постоянства соотношения:

(NH4+ + K+ + Na+ + Ca2+ + Mg2+) / (NO3- + Cl- + SO4^2- + HPO4^2-) = const

Константа больше единицы и специфична для каждой культуры, поскольку растения поглощают катионы в эквивалентно большем количестве, чем анионы.

### Антагонизм

Физиологический эффект смеси солей меньше, чем эффект от каждой соли в отдельности. Катионы разных элементов конкурируют друг с другом при адсорбции на поверхности корней.

Антагонизм сильнее проявляется:
- Между одноимённо заряженными ионами
- Когда концентрация одного иона значительно превышает концентрацию другого

**Установленные пары антагонизма:**

| Элемент 1 | Элемент 2 |
|-----------|-----------|
| Железо (Fe) | Кальций (Ca) |
| Алюминий (Al) | Натрий (Na) |
| Железо (Fe) | Цинк (Zn) |
| Марганец (Mn) | Цинк (Zn) |
| Медь (Cu) | Цинк (Zn) |
| Железо (Fe) | Марганец (Mn) |
| Медь (Cu) | Молибден (Mo) |

### Синергизм

Взаимное усиление физиологического действия элементов. Может наблюдаться как между разнозаряженными, так и между одноимённо заряженными ионами. Чаще проявляется при невысоком содержании элементов в растворе.

**Установленные пары синергизма:**

| Элемент 1 | Элемент 2 |
|-----------|-----------|
| Сера (S) | Марганец (Mn) |
| Сера (S) | Цинк (Zn) |
| Медь (Cu) | Кобальт (Co) |
| Медь (Cu) | Бор (B) |
| Медь (Cu) | Цинк (Zn) |
| Марганец (Mn) | Кобальт (Co) |
| Марганец (Mn) | Молибден (Mo) |
| Медь (Cu) | Молибден (Mo) |
| Медь (Cu) | Марганец (Mn) |
| Кальций (Ca) | Кобальт (Co) |
