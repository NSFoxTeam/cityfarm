## Описание

Классические рецепты гидропонных питательных растворов. Количество солей указано в граммах на 10 литров воды. Для всех рецептов используются соли, очищенные перекристаллизацией.

## Рецепт 1: По Кнопу

Исходное значение pH смеси: ~5.7. В процессе питания растений смесь подщелачивается до ~7.0.

| Компонент | Назначение |
|-----------|------------|
| Кальция нитрат (кальциевая селитра) | Ca, N |
| Калия нитрат (калиевая селитра) | K, N |
| Калия фосфат (монокалийфосфат) | K, P |
| Магния сульфат | Mg, S |
| Калия хлорид | K, Cl |
| Железа хлорид | Fe |

## Раствор микроэлементов (по Хогланду)

Добавляется по 1 мл на каждый литр готового раствора (к рецептам 1, 4 и 5).

| Компонент | Элемент |
|-----------|---------|
| Хлористый литий | Li |
| Сульфат меди | Cu |
| Борная кислота | B |
| Сульфат цинка | Zn |
| Хлористый марганец (двухвалентный) | Mn |

## Важные правила

- Растворять каждую соль последовательно в горячей воде
- Не растворять все соли одновременно в одной емкости -- они могут прореагировать между собой и выпасть в осадок
- Остудить раствор перед заливкой в резервуар
- Измерить электропроводность раствора для определения количества веществ
- Не использовать для гидропоники удобрения, предназначенные для внесения в почву
- Избегать удобрений с азотом в форме мочевины или аммония
- Азотсодержащие удобрения должны включать азот только в нитратной форме
- Все удобрения должны быть полностью растворимы в воде
