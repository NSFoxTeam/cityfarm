## Суть

Электронный pH-метр измеряет уровень кислотности питательного раствора. Основной элемент прибора -- комбинированный электрод со стеклянной мембраной, реагирующей на концентрацию ионов водорода. Правильное хранение электрода и соблюдение алгоритма измерений критически важны для точности.

## Детали

### Устройство pH-метра

| Компонент | Назначение |
|-----------|------------|
| Пластиковый корпус | Защита внутренних компонентов |
| Дисплей | Отображение результатов pH |
| Кнопка вкл/выкл | Управление питанием |
| Кнопка автокалибровки | Запуск процедуры калибровки |
| Электрод | Измерение ионов водорода |
| Крышка | Защита электрода от повреждений и высыхания |

### Комбинированный электрод

- **Стеклянная мембрана** -- реагирует на концентрацию ионов водорода, создаёт электрический потенциал пропорционально уровню pH
- **Токоотводящий электрод** -- передаёт потенциал, изготовлен из серебра
- **Раствор электролита** -- хлорид калия (KCl) между мембраной и токоотводящим электродом
- **Кристаллы KCl** -- обеспечивают насыщение внутреннего раствора электролита
- **Электрод сравнения** -- создаёт стабильный опорный потенциал для измерения разности потенциалов
- **Мембрана с пористой структурой** -- из керамики или полимера, обеспечивает взаимодействие KCl с исследуемой средой

### Хранение электрода

- Хранить в водных растворах, богатых ионами (стеклянная мембрана остаётся насыщенной)
- Между измерениями помещать в раствор электролита (3-молярный раствор KCl)
- При отсутствии раствора для хранения -- буферный раствор с pH 4 или 7
- **Не хранить в дистиллированной воде** -- вытягивает ионы из мембраны, повреждает электрод

### Алгоритм калибровки

1. Промыть электрод дистиллированной водой
2. Поместить в буферный раствор pH 7, дождаться стабилизации, нажать кнопку калибровки
3. Повторить с буферными растворами pH 4 и pH 10
4. Калибровочные буферные растворы идут в комплекте с прибором

**Частота калибровки:** раз в месяц при частом использовании, при замене электрода, при подозрении на некорректность показаний.

### Алгоритм измерения

1. Взять пробу раствора из бака чистой стеклянной посудой, предварительно промытой дистиллированной водой
2. Промыть электрод pH-метра дистиллированной водой
3. Включить pH-метр и дождаться обнуления
4. Опустить электрод в стеклянную посуду с раствором, через минуту считать результат
5. Залить в крышку раствор для хранения и поместить туда электрод

### Альтернативные способы измерения pH

**Жидкий pH-тест (GHE):**
- Содержит индикаторный раствор с органическими соединениями
- Несколько капель добавляются в пробу воды
- Цвет сравнивается с эталонной шкалой (диапазон 4.5--10)

**Лакмусовая бумага:**
- Индикаторный слой, пропитанный лакмусом
- При погружении изменяет цвет в зависимости от pH
- Цвет сравнивается с эталонной шкалой (диапазон 1--14)
