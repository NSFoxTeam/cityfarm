## Суть

EC-метр и TDS-метр -- приборы для измерения концентрации минеральных веществ в питательном растворе. EC-метр измеряет электропроводность (способность проводить ток), TDS-метр -- общее количество растворённых веществ. TDS рассчитывается косвенно на основе EC с коэффициентом пересчёта 500--700.

## Детали

### Различие EC и TDS

| Параметр | EC | TDS |
|----------|-----|-----|
| Что измеряет | Электропроводность | Общее содержание растворённых веществ |
| Единица измерения | мСм/см (mS/cm) | ppm |
| Метод | Прямое измерение | Косвенный расчёт из EC |
| Коэффициент пересчёта | -- | 500--700 |

Оба показателя тесно связаны и дополняют друг друга при анализе качества питательных растворов.

### Устройство EC/TDS-метра

| Компонент | Назначение |
|-----------|------------|
| Влагостойкий корпус | Защита внутренних компонентов |
| Цифровой дисплей | Отображение EC (мСм/см) или TDS (ppm) |
| Кнопка переключения EC/TDS | Выбор единиц измерения |
| Кнопка автокалибровки | Запуск калибровки |
| Электроды | Два электрода из нержавеющей стали, устойчивые к коррозии |
| Датчик температуры | Встроенный, автоматическая температурная коррекция |

### Принцип работы электродов

1. Электроды погружаются в раствор
2. Через них проходит слабый электрический ток благодаря заряженным частицам (ионам: Na+, Cl-, K+)
3. Один электрод -- катод, другой -- анод, создаётся электрическое поле
4. Прибор фиксирует сопротивление между электродами
5. Чем больше ионов, тем ниже сопротивление и выше проводимость
6. EC переводится в TDS с использованием коэффициента пересчёта

### Алгоритм калибровки

1. Включить прибор
2. Промыть электроды дистиллированной водой
3. Погрузить электроды в калибровочный раствор хлорида калия (KCl) с электропроводностью 1413 мкСм/см
4. Дождаться стабилизации показаний на дисплее
5. Нажать кнопку автоматической калибровки
6. Убедиться, что показания соответствуют электропроводности калибровочного раствора

Калибровочный раствор часто поставляется в комплекте с прибором.

### Алгоритм измерения

**Важно:** измерять EC/TDS после добавления всех минеральных элементов в воду.

1. Взять пробу раствора из бака чистой стеклянной посудой, предварительно промытой дистиллированной водой
2. Промыть электроды дистиллированной водой
3. Поместить электроды в питательный раствор
4. Дождаться стабилизации показаний на дисплее
5. При нескольких пробах -- повторить цикл с первого шага
6. По окончании протереть электроды и закрыть крышкой
