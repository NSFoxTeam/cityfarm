## Суть

Кондиционер -- устройство для изменения параметров воздуха в помещении: температура, влажность, чистота, скорость циркуляции. Для сити-ферм используются сплит-системы трёх уровней: настенные (дом/офис), полупромышленные (средние помещения), промышленные мультизональные (большие площади).

## Детали

### Классификация сплит-систем

| Тип | Мощность | Применение | Особенности |
|-----|----------|------------|-------------|
| Настенная | до 7 кВт | Дом, офис, малая ферма | Компактная, на стене |
| Напольно-потолочная | > 7 кВт | Магазины, средние помещения | Поток в одну сторону |
| Кассетная | > 7 кВт | Офисы (подвесной потолок) | Поток в 4 стороны, тихая работа |
| Канальная | > 7 кВт | Несколько помещений | Через воздуховоды, равномерное распределение |
| Мультизональная | промышл. | Большие площади | Трассы до 150 м, разные внутренние блоки |

### Старт-стопные vs инверторные

| Параметр | Старт-стопный | Инверторный |
|----------|---------------|-------------|
| Принцип | Вкл/выкл компрессора | Переменная скорость |
| Скачки мощности | Да | Нет |
| Точность температуры | Низкая | Высокая |
| Энергопотребление | Выше | Ниже |
| Износ | Больше | Меньше |
| Цена | Ниже | Выше |
| Ремонт | Проще | Сложнее |
| Применение | Склады, офисы | Фермы, жилые помещения |

**Рекомендация для начального этапа:** 1-3 сплит-системы для небольшого помещения сити-фермы.

### Рекомендуемые бренды
- **Рекомендуется:** MDV, Gree (высокое качество, надёжность)
- **Не рекомендуется:** AUX (бюджетная техника), Haier (чувствителен к перепадам напряжения)
- **Не покупать в:** Leroy Merlin, DNS, Ozon -- заниженные характеристики по производительности
