# Hydroponics Feed Dataset

**Источник:** Kaggle
**Ссылка:** https://www.kaggle.com/datasets/pranav941/hydroponics-feed

## Описание

Данные с IoT-датчиков гидропонной системы: температура, влажность, heat index. Реальные сенсорные данные.

## Применение для CityFarm

- Прямая аналогия с нашими DHT22 (температура + влажность)
- Обучение моделей предсказания микроклимата
- Детекция аномалий в сенсорных данных
