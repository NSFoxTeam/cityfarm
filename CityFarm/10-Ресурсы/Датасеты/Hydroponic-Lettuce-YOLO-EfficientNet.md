# YOLO-EfficientNet: Growth Status of Hydroponic Lettuce

**Источник:** PMC (Horticulturae journal)
**Ссылка:** https://pmc.ncbi.nlm.nih.gov/articles/PMC10857641/

## Описание

Определение стадии роста гидропонного салата через YOLO + EfficientNet. Классификация фаз развития по изображениям.

## Применение для CityFarm

- Автоматическое определение фазы роста салата через камеру
- Привязка фазы роста к оптимальным параметрам из наших карточек
- Замкнутый цикл: камера → фаза роста → автокоррекция pH/EC/света
