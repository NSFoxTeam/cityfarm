# Smart Farming Sensor Data for Yield Prediction

**Источник:** Kaggle
**Ссылка:** https://www.kaggle.com/datasets/atharvasoundankar/smart-farming-sensor-data-for-yield-prediction

## Описание

Сенсорные данные smart farming для предсказания урожайности. Связь параметров среды с результатом.

## Применение для CityFarm

- Модель предсказания урожайности по данным наших датчиков
- Корреляция pH/EC/температура/свет → yield
- Оптимизация параметров для максимального выхода
