## Суть

Выбор субстратов и упаковки для гидропонной продукции влияет на экологический след предприятия. Натуральные субстраты (джут, кокос, лён) компостируются, синтетические (пенный пластик) -- нет. Упаковку рекомендуется выбирать из мономатериалов, пригодных к переработке.

## Детали

### Субстраты по экологичности

| Субстрат | Материал | Утилизация | Примечание |
|----------|----------|------------|------------|
| Синтетическая губка | Пенный пластик | Не перерабатывается | Загрязнена после использования |
| Джутовый коврик | Натуральный джут | Компостирование | Оптимален при наличии компостера |
| Минеральная вата | Базальтовая порода | Повторное использование, затем компостирование | Производство: нагрев до 1500 C + центрифугирование |
| Торфяные субстраты | Природные элементы | Компостирование | Смесь из природных элементов |
| Кокосовый субстрат | Кокосовая стружка | Компостирование | Высокий транспортный след (южные страны) |
| Льняные коврики | Волокно льна/хлопка | Компостирование | Для микрозелени; только натуральная ткань |

### Упаковка для продукции

**Салат в горшочках:**
- Горшочек из полипропилена (маркировка 5) -- перерабатывается
- Пакет-конус из полипропилена или полиэтилена -- минимум краски для переработки

**Микрозелень:**
- Прозрачный полипропиленовый контейнер с крышкой -- перерабатывается в большинстве городов

**Салаты под срез:**
- Контейнер из прозрачного полипропилена
- Флоупак: только мономатериал (один вид пластика) для возможности переработки

**Поставки в рестораны:**
- Оборотные герметичные контейнеры из пищевого пластика
- Экспедитор забирает контейнеры предыдущей поставки

### Маркировка пластика
- Маркировка 1 (ПЭТ): бутылки из-под напитков -- лучше всего перерабатывается
- Маркировка 2 (ПНД): тара из-под бытовой химии
- Маркировка 4 (ПВД): пластиковые пакеты
- Маркировка 5 (ПП): белые и прозрачные контейнеры, вёдра
