## Суть

Выбор субстрата и упаковки для сити-фермы с точки зрения экологичности. Натуральные субстраты (джут, торф, лен) можно компостировать, синтетические (губка) требуют утилизации. Для упаковки предпочтительны мономатериалы из полипропилена, которые принимают на переработку.

## Детали

### Экологичность субстратов

| Субстрат | Экологичность | Утилизация |
|----------|---------------|------------|
| Синтетическая губка (полиуретан) | Низкая | Не перерабатывается, загрязнена |
| Джутовый коврик | Высокая | Натуральный материал, можно компостировать |
| Минеральная вата (агровата) | Средняя | Можно повторно использовать, затем компостировать. Производство: нагрев базальтовой породы до 1500 °C и центрифугирование |
| Торфяные субстраты | Высокая | Природные элементы, можно компостировать |
| Кокосовый субстрат | Средняя | Можно компостировать, но высокий транспортный след (кокосы растут только в южных странах) |
| Льняные коврики | Высокая | Если ткань натуральная (лен/хлопок), можно компостировать |

**Рекомендация:** если ферма за городом и есть место -- установить компостер для переработки пищевых отходов и натуральных субстратов в удобрение.

### Упаковка для продукции

#### Салат в горшочках
- Горшочек из полипропилена (маркировка 5) -- можно сдать на переработку
- Пакет-конус из мягкого полипропилена или полиэтилена -- чем меньше краски, тем лучше для переработки
- Обозначать возможность переработки на упаковке

#### Микрозелень
- Полипропиленовый прозрачный контейнер с крышкой (маркировка 5)
- Принимают на переработку в любом крупном городе

#### Салаты под срез (салатные миксы)
- Контейнер: прозрачный полипропилен
- Флоу-пак: выбирать мономатериал (один вид пластика -- полипропилен или полиэтилен)
- Мономатериал имеет шанс попасть на переработку; композитные материалы -- нет
- Не использовать много краски

#### Оборотная тара (для ресторанов)
- При регулярных поставках -- использовать герметичные контейнеры из пищевого пластика
- Экспедитор забирает контейнеры с предыдущего привоза
- Исключает одноразовую упаковку
