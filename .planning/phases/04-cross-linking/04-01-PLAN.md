  <action>
Create a bash validation script at `.planning/phases/04-cross-linking/validate-links.sh` that:

1. **Counts cross-folder links**: Extracts all `[[target]]` wiki-links from each .md file in CityFarm/, resolves the target to a file, determines if source and target are in different folders, reports total intra-folder vs cross-folder counts.

2. **Detects broken links**: Extracts all `[[target]]` and `[[target|display]]` patterns, strips display text, checks if a matching .md file exists anywhere in CityFarm/. Reports any broken links with source file and target.

3. **Checks for missing ## Связанные заметки sections**: Scans all content notes (excluding MOC files, Home.md, Glossary.md, _templates/) for the presence of `## Связанные заметки`. Lists notes missing it.

4. **Summary output**: Prints totals — total links, intra-folder, cross-folder, cross-folder ratio %, broken links count, notes missing related section.

The script should accept CityFarm/ path as argument (default: `CityFarm/`). Make it executable.

Before creating the script, build a complete index of all note filenames (without .md) in CityFarm/ — this index will be used both by the script and mentally by subsequent tasks to ensure wiki-link targets are exact filename matches.
