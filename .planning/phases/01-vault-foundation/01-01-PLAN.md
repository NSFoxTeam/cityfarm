  <action>
Create three Templater templates in CityFarm/_templates/. Each uses Templater tp.file and tp.date functions in YAML frontmatter. NOTE: Do NOT use tp.frontmatter (known issue #1290 -- doesn't work on file creation). Use raw YAML with Templater expressions directly.

1. **knowledge-note.md** -- for atomic knowledge notes:
```yaml
---
title: "<% tp.file.title %>"
type: knowledge
topic: ""
tags: []
source: ""
created: <% tp.date.now("YYYY-MM-DD") %>
status: draft
---
```
Body sections: ## Суть, ## Детали, ## Связанные заметки

2. **reference-table.md** -- for parameter reference tables:
```yaml
---
title: "<% tp.file.title %>"
type: reference
topic: ""
parameters: []
crops: []
created: <% tp.date.now("YYYY-MM-DD") %>
status: draft
---
```
Body sections: ## Параметры (with markdown table scaffold: Параметр | Диапазон | Единица | Примечание), ## Источники

3. **parameter-card.md** -- for individual IoT-mappable parameters:
```yaml
---
title: "<% tp.file.title %>"
type: parameter
parameter: ""
unit: ""
optimal_range: ""
sensor: ""
alert_low: null
alert_high: null
created: <% tp.date.now("YYYY-MM-DD") %>
status: draft
---
```
Body sections: ## Описание, ## Оптимальные значения (table by crop), ## IoT-маппинг (sensor, threshold, alert, action), ## Источники
  </action>
