  <files>CityFarm/Glossary.md</files>
  <action>
Create CityFarm/Glossary.md with YAML frontmatter and structured term tables for STT error correction.

Frontmatter:
```yaml
---
title: Глоссарий гидропонных терминов
type: reference
purpose: stt-correction
created: 2026-03-09
---
```

Structure the glossary with these categories (use markdown tables with columns: Правильно | STT-варианты | English | Категория):

1. **Элементы питания** -- macro and micro nutrients:
   - Макроэлементы: азот/nitrogen, фосфор/phosphorus, калий/potassium, кальций/calcium, магний/magnesium, сера/sulfur
   - Микроэлементы: железо/iron, марганец/manganese, молибден/molybdenum, цинк/zinc, бор/boron, медь/copper
   - Include observed STT errors from module_03 transcripts: азод->азот, кали->калий, фосфар->фосфор, кальцы->кальций, магни->магний, молевден/молебден->молибден, цинг->цинк

2. **Гидропонные системы** -- growing system types:
   - гидропоника, аэропоника, NFT, DWC, капельный полив, субстрат, рециркуляция, дренаж
   - STT errors: гидрапоника, аеропоника, субстрад, рецеркуляция/рициркуляция

3. **Параметры среды** -- environmental parameters:
   - электропроводность/EC, кислотность/pH, температура, влажность, освещённость/PAR/PPFD, CO2
   - STT errors: электропровадность, фотосинтэз, транспирацы

4. **Оборудование и компоненты** -- equipment:
   - помпа, таймер, датчик, контроллер, LED-светильник, спектр, рефлектор
   - Common terms from hardware context

5. **Культуры и растения** -- crops:
   - салат, базилик, руккола, укроп, петрушка, микрозелень, шпинат

6. **Общие термины курса** -- general course terms:
   - доцент (дациент/дацент), агрохимия (агрохими/агра химия), органические (органическе), минеральные (менеральные)

Add a header note explaining the glossary's purpose: domain-specific terms for correcting STT (speech-to-text) errors in course transcriptions. Note that STT-варианты column contains observed errors from transcripts that should be replaced with the correct term during processing.

Target: ~60-80 terms across all categories. This is a starting glossary; it will be expanded during Phase 2 processing as new STT errors are discovered.
