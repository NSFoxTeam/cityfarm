# CityFarm Knowledge Base

## What This Is

Структурированная база знаний (Obsidian vault) по сити-фермерству и гидропонике, созданная из материалов курса "Сити-фермер: от идеи до PRO". Vault живёт внутри проекта CityFarm (`CityFarm/`) и служит справочником по агротехнике, а также проектной документацией для IoT-платформы CityFarm.

## Core Value

Вся информация по сити-фармингу структурирована по темам и доступна как справочник для принятия решений по проекту CityFarm (какие параметры мониторить, какие пороги ставить, как реагировать на отклонения).
